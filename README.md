Aplicação Web para Monitoramento de Riscos Costeiros

Este repositório contém o projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) do Curso Técnico em Desenvolvimento de Sistemas – EaD, com o objetivo de monitorar riscos costeiros no litoral paulista por meio de mapas, gráficos, indicadores e relatórios interativos.

Funcionalidades

Mapa interativo

Tela de login

Painel principal com indicadores

Alertas de eventos extremos

Consulta de dados históricos

Visualização gráfica (Chart.js e Plotly.js)

Exportação de relatatórios em PDF e CSV

Tela de contato e sugestões

Modo acessível (alto contraste e fonte ampliada)

Tecnologias Utilizadas

Front-end: HTML, CSS, JavaScript, Leaflet.js, Chart.js, Plotly.js
Back-end: PHP, MySQL
Manipulação de dados: PapaParse, Danfo.js, Math.js
Ferramentas: VS Code, XAMPP, Git, GitHub

Como Rodar o Projeto

Clone o repositório:

git clone https://github.com/liviazanichelli/monitoramento_costeiro.git


Mova o projeto para o diretório do XAMPP:

C:\xampp\htdocs\


Inicie os serviços Apache e MySQL no XAMPP.

Abra o phpMyAdmin e importe o arquivo .sql do projeto.

Acesse pelo navegador:

http://localhost/monitoramento_costeiro/
